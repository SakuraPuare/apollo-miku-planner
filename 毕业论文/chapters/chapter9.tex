\chapter{总结与展望}

\section{研究工作总结}

本文以百度Apollo 11.0开源自动驾驶平台为研究对象，针对其路径规划模块在多障碍物密集场景下的决策失效问题，从源码分析、问题识别、算法设计到仿真验证进行了系统性研究，提出多障碍物交互运动学统一规划算法（Multi-obstacle Interaction-density Kinematic-prediction Unified Planner，MIKU）。主要完成了以下工作。

\begin{enumerate}
\item \textbf{对Apollo Planning模块进行了源码级深度分析}\quad{}系统梳理了Planning模块的Scenario-Stage-Task三级管理架构、PublicRoadPlanner中基于EM Planner思想的路径-速度解耦两阶段优化流程、ReferenceLine平滑机制、PathDecider决策逻辑、PathBoundsDecider的路径边界构建流程以及SpeedBoundsDecider的ST边界构建机制。通过对关键源码的逐行分析，揭示了PathBoundsDecider在路径边界构建阶段的逐障碍物独立决策模式、统一安全裕度分配、动态障碍物排除机制，以及SpeedBoundsDecider与路径阶段之间单向且有限的信息传递特征，为后续问题识别和算法改进提供了可靠依据。

\item \textbf{从三个维度系统识别了多障碍物场景下路径规划失效的根本瓶颈}\quad{}通过源码分析从安全距离、决策粒度和时空协调三个维度识别了三个核心问题。问题一，所有障碍物使用统一的安全裕度$\delta$，不区分威胁程度，低威胁障碍物不必要地压缩了通行带宽度，加剧了路径阻塞的发生概率。问题二，PathBoundsDecider对每个障碍物独立执行贪心避让方向选择，当多个障碍物分布在参考线两侧时，独立决策产生的矛盾避让方向导致路径边界上下界交叉，即$l^+ < l^-$，触发BLOCKED状态，迫使车辆执行不必要的停车，是导致规划失败的直接瓶颈。问题三，动态障碍物在路径决策阶段被完全排除，即源码中通过\texttt{IsStatic()}过滤，使得路径边界无法反映动态障碍物的实际影响，且路径-速度之间缺乏时空约束传递通道，导致速度阶段只能基于静态快照做出过度保守的YIELD停车决策。三个问题的共同根源在于路径边界构建缺乏全局视野、分级机制和时间信息。

\item \textbf{建立了多障碍物统一规划的数学模型}\quad{}将多障碍物避让方向选择建模为组合优化问题$(\mathbf{d}^*, \tau^*) = \arg\max \min_s W(s, \mathbf{d}, \tau)$。通过引理1（最优分割的连续性），利用反证法证明了最优避让方向分配在障碍物横向排序中具有连续性，将搜索空间从$O(2^k)$降至$k+1$个候选；通过定理1（最大间隙最优性），直接代入验证了每个候选分割的通行带宽度恰好等于对应的横向间隙，将最优通行带的精确求解复杂度降为$O(k \log k)$。该数学框架为工程实现提供了严格的理论保证。

\item \textbf{设计并实现了基于预测轨迹的时变障碍物分组算法}\quad{}在SL层，提出多因子威胁度评估模型，结合TTC、横向重叠率、相对速度、障碍物类型和交互密度五个因子量化威胁程度，实现差异化安全裕度$\delta_i$的分配；通过扫描线算法检测障碍物的纵向重叠关系，将相互关联的障碍物聚为连通分量，在组内按横向位置排序并计算最大间隙，执行组级全局最优避让方向决策。在ST层，引入基于二阶运动学模型的到达时间函数$\tau(s)$和预测轨迹，将动态障碍物纳入路径边界决策；将时变通行带的时间有效性条件投影至ST空间，构建时空可通行走廊，其边界约束与SpeedBoundsDecider的ST边界在QP层面融合，建立了此前缺失的路径-速度约束传递通道。算法总复杂度为$O(n \log n)$，满足实时性要求。

\item \textbf{通过仿真实验全面验证了改进算法的性能}\quad{}基于全链路Python仿真平台设计了4个测试场景，覆盖单一动态障碍物、动静混合、多动态障碍物以及密集静态障碍物四类典型工况，从通行成功率、平均速度、最大加速度幅值、最大jerk幅值、到达时间和QP求解耗时六个维度进行Baseline与MIKU的定量对比。实验结果表明，MIKU将平均巡航速度从\MSummaryBaselineAvgV\,m/s提升至\MSummaryMikuAvgV\,m/s，提升\MSummarySummaryVImprovement\%；通行成功率从\MSummaryBaselinePassCount{}个场景提升至\MSummaryMikuPassCount{}个场景；轨迹平顺性指标$|a|_{max}$与$|j|_{max}$在所有通行场景中均低于Baseline；QP求解总耗时由Baseline的\MSummaryBaselineQpTotal\,ms降至\MSummaryMikuQpTotal\,ms，降幅约89\%，原因在于路径阶段提前避开冲突使速度阶段无需重度回退，整体满足Apollo的100\,ms规划周期要求。消融实验表明C5走廊注入在简单场景为鲁棒性保险阀而非常态收益项，复杂场景下方为正贡献。
\end{enumerate}

\section{主要创新点}

本文的主要创新点归纳如下。

\begin{enumerate}
\item \textbf{将多障碍物避让方向选择建模为最大通行带宽度优化问题，并给出了最大间隙策略最优性的理论分析}\quad{}引理1通过反证法证明了最优避让方向分配在障碍物横向排序中具有连续性，将指数级搜索空间降维为线性枚举；定理1进一步证明了每个候选分割的通行带宽度恰好等于对应的横向间隙，从而将最优避让策略等价为选择最宽间隙这一直观操作。该理论结果不仅提供了$O(k \log k)$的精确求解算法，也从数学上解释了为何Apollo原始贪心策略在多障碍物场景下必然失效。

\item \textbf{提出了基于扫描线的时变障碍物分组算法}\quad{}通过纵向重叠检测将互相关联的障碍物聚为连通分量，在组级别应用全局最优决策，从根本上消除了逐障碍物贪心策略产生矛盾避让方向的问题。分组机制使得不同连通分量的决策互不干扰，在保证全局最优性的同时保持了算法的局部性和可扩展性。

\item \textbf{设计了面向通行带优化的威胁度分级与动态安全距离机制}\quad{}基于多因子评估方法，结合TTC、横向重叠率、相对速度、障碍物类型和交互密度五个因子量化威胁程度，据此分配差异化的安全裕度$\delta_i$。该机制的核心贡献不在于威胁度模型本身，而在于将差异化$\delta_i$与最大间隙算法相结合，引理1和定理1对$\delta$是否为常数不做假设，因此差异化$\delta_i$可直接代入最优求解框架。该设计使低威胁障碍物释放通行空间，在不降低安全性的前提下扩大通行带宽度。

\item \textbf{引入到达时间函数和预测轨迹，将动态障碍物纳入路径边界决策，并建立ST走廊与速度规划的协调机制}\quad{}弥补了Apollo PathBoundsDecider在路径边界构建阶段完全排除动态障碍物的不足，利用预测轨迹计算障碍物在自车到达时刻的预期位置，使路径边界能够反映动态障碍物的时变特性。对于正在横穿的行人等场景，时变投影能够捕捉到障碍物移开后间隙打开的可能性，避免过度保守的停车决策。进一步将时变通行带投影至ST空间，揭示其本质上构成时空可通行走廊，走廊边界转化为速度QP的上界约束，与SpeedBoundsDecider的ST边界形成双源融合，在不修改速度规划模块结构的前提下实现了路径-速度协调，解决了第六章所揭示的路径-速度信息断层问题。
\end{enumerate}

\section{未来研究方向}

本文工作在多个层面仍存在进一步拓展的空间，未来研究可沿以下方向展开。

\begin{enumerate}
\item \textbf{交互式预测与博弈感知决策}\quad{}本文的时变障碍物投影建立在Prediction模块单次前向输出的预测轨迹之上，已能够刻画行人横穿、车辆并线等典型动态过程，但尚未显式建模自车决策对周车行为的反作用。在密集交互场景中，障碍物的实际轨迹会随自车的避让方向与加减速节奏发生改变，开环预测因此存在固有偏差。后续可引入基于博弈论的交互决策方法，例如迭代最优响应，将自车与周车建模为双向耦合系统，把自车候选行为反馈进预测过程形成闭环交互推断；进一步将差异化威胁度$\delta_i$从静态评估扩展为交互均衡下的动态量，使最优分配同时反映几何约束与交互代价。

\item \textbf{到达时间与速度规划的迭代优化}\quad{}本文采用的二阶运动学到达时间模型相较匀速假设已有显著改进，在大多数典型工况下能够保证时空对齐精度，但仍属于单次前向估计；当QP输出的速度曲线与$\tau(s)$估计偏离较大时，例如需要大幅减速的场景，时变投影可能产生偏差。后续可将到达时间估计从单次前向计算改为路径与速度的迭代求解，首次以匀速假设计算通行带，基于该通行带生成路径和速度曲线后，用更新的速度剖面重新估计到达时间，重复迭代至收敛。研究重点在于迭代收敛性的理论证明与计算预算控制，使算法在保证实时性的同时获得更精确的时空匹配。

\item \textbf{纵向部分重叠的精确处理}\quad{}本文对$s$方向仅部分重叠的障碍物组采用公共截面间隙作为通行带宽度的保守估计，该处理在障碍物纵向间距较小时是合理的，但在某些特殊几何构型下未能完全释放可用空间。后续可将组级决策扩展为沿$s$方向逐截面求解，离散化生成多组间隙序列后在序列上求全局最优分配，更精确地刻画通行带沿纵向的变化特征。该方向的核心挑战在于精度提升与计算开销之间的折衷，需要研究自适应离散粒度策略与分支限界加速技术。

\item \textbf{学习辅助的间隙选择}\quad{}当多个候选间隙宽度接近时，最大间隙策略虽在几何意义上最优，但未显式建模变道代价、上下游道路拓扑、驾驶习惯等高层因素。后续可引入学习方法，例如基于历史驾驶数据训练的间隙评估网络，对候选间隙进行综合评分，在宽度近似的情况下做出更符合人类驾驶偏好的选择，实现几何最优与体验最优的统一。引理1与定理1的结论对$\delta_i$的具体形式不做假设，因此学习模块可以以即插即用的方式与最大间隙框架协同工作。

\item \textbf{实车平台验证与工程化部署}\quad{}本文实验基于Apollo PNC仿真平台完成，已系统覆盖单一动态、动静混合、多动态以及密集静态四类典型工况。下一阶段计划将改进算法部署至搭载Apollo系统的实车平台，在真实交通环境中评估传感器噪声、感知延迟与执行误差对算法的影响，并完善针对极端工况的降级策略与失效安全机制。重点研究感知误差在路径边界构建中的传播规律，以及规划-控制延迟下时变通行带的鲁棒性，推动改进算法从仿真验证走向规模化应用。
\end{enumerate}
