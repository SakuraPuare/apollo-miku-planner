# 毕业论文 docx 排版规范

依据为学校《规范化要求（2025 年 12 月修订）》与 0045robot 检测器实际口径。

## 1. 全局默认

| 项 | 值 |
|---|---|
| 页面 | A4，上下 2.54 cm，左右 3.17 cm |
| 行距 | 1.5 倍，参考文献除外 |
| 中文字体 | 宋体，允许宋/黑/仿宋/楷，其余一律回落宋体 |
| 西文字体 | Times New Roman |
| 颜色 | 全文黑色，含超链接 |
| 自动项目编号 | 禁用 |
| 页脚 | 五号 TNR 居中 |
| 页码 | 前置页罗马 I/II，正文阿拉伯从 1 |

## 2. 章节块识别顺序

cover → declaration（学生声明）→ authorization（授权使用说明）→ toc（目录）→ zh_abstract（中文题目起）→ en_abstract（英文题目起）→ body（1 绪论起）→ bibliography（参考文献）→ acknowledge（致谢）→ achievement（本科期间的学习与科研成果）。

### 3.4 参考文献块

| 段 | 对齐 | 字体 | 字号 | 加粗 | 行距 |
|---|---|---|---|---|---|
| "参考文献"标题 | 左 | 黑体 | 四号 | ✓ | 1.5 倍 |
| [N] 条目 | 两端 | 宋体/TNR | 小四 | — | **固定 20 磅 exact** |

条目悬挂缩进。仅此块用 exact 20 pt，成果页 [N] 条目用正文块 1.5 倍。

### 3.5 致谢块

"致 谢"标题居中、黑体、三号、加粗；正文两端、宋体、小四、1.5 倍。

### 3.6 成果页块

同正文段规则。标题"本科期间的学习与科研成果"居中。

## 4. Run 级规则

1. 中文 eastAsia 仅允许宋/黑/仿宋/楷，其余回落宋体，防 Dengxian、Calibri、等线泄漏
2. ascii、hAnsi 统一 Times New Roman
3. Consolas、SourceCode、VerbatimChar 在代码块外回落正文字体
4. 段内数学对象 m:r 字号 sz=24，小四
5. 中文段内半角逗号 `,` 转全角`，`，参考文献英文逗号保留
6. 正文段 run 级 `<w:b/> <w:bCs/>` 一律清除，caption、heading、前缀段例外
7. 居中 16pt 段首 run 补 `<w:b/>`，防题目"部分常规"误报
8. "目  录"段 run 补 `<w:b/>`
9. caption 前缀"图/表/代码/算法"与 N.M 之间清零全部空白，含 U+00A0 与全角空格

## 5. 检测器硬性口径

| 项 | 口径 |
|---|---|
| 5.1 | 图/表/代码/算法 caption 前缀与数字无空格 |
| 5.2 | 代码/算法 caption 段无首行缩进 |
| 5.3 | 目录段段后 0，非 Word 默认 10 pt |
| 5.4 | 目录段 1.5 倍行距，非单倍 |
| 5.5 | TOC1 无首行缩进，TOC2/3 保留 2 字符体现层级 |
| 5.6 | 摘要页"论文题目"段段前 6 pt 段后 1.54 pt，非摘要前缀段 |
| 5.7 | 参考文献行距固定 20 磅 exact，非 1.5 倍 auto |
| 5.8 | 独立公式编号段右对齐 |
| 5.9 | 正文段任何 `<w:b/>` 即被判加粗，须在 run 级清除 |
| 5.10 | TOC1 所有 run 必须加粗，缺一即报"部分常规" |

## 6. 样式级约束，styles.xml

| 样式 | 关键属性 |
|---|---|
| docDefaults rFonts | ascii/hAnsi/cs = Times New Roman，eastAsia = 宋体 |
| Heading1 | sz=28，b=true，eastAsia=黑体 |
| Heading3 | sz=24，b=true，eastAsia=宋体 |
| Bibliography | line=400，lineRule=exact |
| TOC1 | after=0，line=360 auto，firstLine=0，tab right dot pos=8306，b=true，eastAsia=黑体，sz=24 |
| TOC2 | after=0，line=360 auto，firstLineChars=200 firstLine=480，tab right dot pos=8306，b=false，eastAsia=宋体，sz=24 |

Heading2 sz 当前为 32，§规范要求小四 12pt（sz=24），待核对。

## 7. 检测器提醒级，不强制

DOI 格式合法即可，内容人工核对；TOC 域页码在 Word 首次更新时同步，检测器读的是缓存值；分页孤行、短表续表、公式周围空格均为提醒级。
