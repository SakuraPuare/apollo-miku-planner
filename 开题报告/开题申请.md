## 三、开题申请理由

### 3.1 选题背景与意义

随着人工智能和自动驾驶技术的快速发展，路径规划作为自动驾驶系统的核心模块，其性能直接影响车辆的安全性和通行效率。在城市道路多障碍物场景下，传统路径规划算法面临计算复杂度高、实时性差、避障策略保守等挑战。

百度Apollo作为全球领先的开源自动驾驶平台，其Planning模块采用基于凸优化的路径规划方法，具有较强的工程实践价值。然而，现有算法在多障碍物密集场景下存在ST边界重叠、PathDecider决策保守、场景切换滞后等问题。本课题旨在针对上述问题进行深入研究并提出改进方案，具有重要的理论意义和实践价值。

### 3.2 研究目标

1. 深入分析Apollo Planning模块的架构设计和核心算法；
2. 识别现有算法在多障碍物场景下的具体问题；
3. 设计并实现基于动态权重的多障碍物ST边界优化算法；
4. 通过仿真实验验证改进算法的有效性。

### 3.3 研究内容

1. **Apollo Planning模块分析**：研究Routing与Planning模块的交互机制，分析PublicRoadPlanner路径-速度解耦两阶段优化（源自早期EM Planner思想）、Lattice Planner等核心算法的实现原理；
2. **多障碍物场景问题识别**：通过源码分析和仿真实验，定位PathDecider、Speed Bounds Decider等模块在多障碍物场景下的性能瓶颈；
3. **改进算法设计与实现**：设计基于动态权重的ST边界优化算法，引入障碍物威胁度评估机制，优化边界构建逻辑；
4. **算法验证与对比分析**：设计仿真实验场景，对比改进算法与原有算法的性能表现。

### 3.4 技术路线

文献调研 → Apollo源码分析 → 问题识别与分析 → 改进算法设计 → 算法实现与调试 → 实验验证 → 论文撰写

### 3.5 预期成果

1. 完成Apollo Planning模块的深入分析报告；
2. 实现基于动态权重的多障碍物ST边界优化算法；
3. 完成仿真实验验证，形成对比分析报告；
4. 撰写符合学院要求的毕业论文。
